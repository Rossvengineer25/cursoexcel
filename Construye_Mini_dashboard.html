01 · El desafío

De los datos al dashboard

Un dashboard (tablero de control) es una vista consolidada de indicadores clave. En este proyecto integraremos todas las herramientas aprendidas.

¿Qué vamos a construir? Un mini dashboard que muestre:

  • Ventas totales (KPI principal)
  • Ventas por categoría (gráfico de barras)
  • Tendencia mensual (gráfico de líneas)
  • Filtros interactivos con segmentadores

Todo alimentado desde una tabla oficial que alimenta una tabla dinámica, la cual es la fuente de los gráficos dinámicos.

02 · Paso 1

Tabla oficial: la base de todo

Convertimos nuestros datos en una tabla oficial para que sean dinámicos y referenciables.

¿Por qué una tabla oficial? Es el cimiento de cualquier dashboard. Al usar Ctrl+T, nuestros datos se vuelven "inteligentes": se expanden automáticamente, usan referencias estructuradas y son el origen perfecto para tablas dinámicas.

💡 Buen hábito: Nombra tu tabla (ej: Ventas) para usarla fácilmente en fórmulas y como origen de tablas dinámicas.

03 · Paso 2

Tabla dinámica: el motor del dashboard

Resumimos los datos de la tabla oficial para alimentar nuestros gráficos.

¿Qué hacemos? Creamos una tabla dinámica a partir de la tabla oficial. Esta tabla dinámica será la fuente única de todos nuestros gráficos dinámicos, asegurando coherencia y facilidad de actualización.

Configuración sugerida:

  • Filas: Categoría y Fecha (agrupada por mes)
  • Valores: Suma de Importe
  • Columnas: Región (opcional)
💡 Consejo: Coloca la tabla dinámica en una hoja nueva llamada "Datos Dashboard" para mantener el orden.
EstructuraFilas: Categoría → Valores: Suma de Importe
04 · Paso 3

Segmentadores: filtros visuales

Añadimos interactividad con segmentadores conectados a la tabla dinámica.

¿Qué logramos? Los segmentadores permiten filtrar la tabla dinámica (y por ende los gráficos) de forma visual e intuitiva. Ideal para que los usuarios del dashboard exploren los datos por sí mismos.

Segmentadores recomendados:

  • Región (Norte, Sur, Este, Oeste)
  • Categoría (Electrónica, Papelería, Hogar)
  • Vendedor (para análisis por persona)
💡 Buen hábito: Coloca los segmentadores en una fila superior, con un diseño limpio y etiquetas claras. Conecta todos los segmentadores a la misma tabla dinámica.
ConexiónSegmentador "Región" → Tabla dinámica "VentasResumen"
05 · Paso 4

Gráficos dinámicos: la visualización

Creamos gráficos que se actualizan al filtrar con los segmentadores.

¿Cómo se hace? Insertamos gráficos dinámicos basados en la tabla dinámica. Cada gráfico debe estar vinculado a la misma tabla dinámica para que los segmentadores los afecten a todos simultáneamente.

Gráficos sugeridos para el dashboard:

  • Gráfico de barras: Ventas por categoría
  • Gráfico de líneas: Evolución mensual de ventas
  • Gráfico circular (opcional): Participación por región
💡 Buen hábito: Usa colores consistentes y etiquetas claras. Asegúrate de que todos los gráficos compartan el mismo origen de datos.
OrigenTabla dinámica: VentasResumen → Gráfico dinámico de barras

07 · Buenas prácticas

Recomendaciones para tu dashboard

Consejos clave para que tu dashboard sea profesional, útil y fácil de mantener.

✔

Usa una sola tabla dinámica

Centraliza todos los gráficos en una misma tabla dinámica para que los filtros afecten a todo el dashboard de forma consistente.

✕

Evita gráficos 3D

Los gráficos 3D distorsionan la percepción de los datos. Opta por gráficos 2D limpios y fáciles de leer.

▦

Diseño ordenado

Organiza los elementos en un layout claro: KPIs arriba, gráficos principales en el centro y filtros en la parte superior.

⟲

Actualización automática

Configura la tabla dinámica para que se actualice al abrir el archivo o usa el botón "Actualizar" para refrescar los datos.

🔍

Prueba la interactividad

Antes de compartir tu dashboard, prueba todos los segmentadores y verifica que todos los gráficos respondan correctamente.
